Quality Corner Features

Wiki

This links to the Altassian Net Wiki, which allows users to write up pages (like OneNote), as well as link between pages and include external links and useful information. This searchable database is editable by all users, just make an account using a non-KH email and you are all set!

Project Tracker

The project tracker is similar to JIRA, but made more simple to save you time. This lets you track progress on tasks and monitor the quality control and review process. Use the tab at the top of the site to access the project tracker!
